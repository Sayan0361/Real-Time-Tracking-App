## Features
- Real-time GPS tracking using `navigator.geolocation`.
- Live updates via Socket.IO.
- Interactive Leaflet/Mapbox map.
- Multi-user support.

## Tech Stack
- Frontend: EJS, JavaScript, Leaflet/Mapbox
- Backend: Node.js, Express, Socket.IO
